# 🎨 UI/UX 设计师 - 设计方案

## 界面布局
采用**沉浸式中心聚焦布局**，减少视觉干扰，确保用户注意力集中在数字上。

1.  **顶部状态栏 (Top Bar)**：
    *   左侧：当前关卡/等级 (Level)。
    *   右侧：最高分记录 (Best Score) 与 设置图标。
    *   中间（游戏进行中）：倒计时进度条（细线型，不抢占视觉重心）。
2.  **核心展示区 (Main Stage)**：
    *   占据屏幕 60% 垂直空间。
    *   **记忆阶段**：数字以大字号居中显示，背景纯净。
    *   **输入阶段**：数字消失，显示输入框（掩码或明文可选）或 数字序列气泡。
3.  **操作控制区 (Control Zone)**：
    *   底部 30% 区域。
    *   **记忆阶段**：仅显示“准备好了”提示或无按钮（避免误触）。
    *   **输入阶段**：弹出自定义数字键盘（0-9, 确认，删除），按键尺寸符合拇指热区。
4.  **结果弹窗 (Result Modal)**：
    *   半屏弹窗，展示本次得分、准确率、耗时，以及“再来一次/下一关”按钮。

## 配色方案
采用**深色专注模式 (Dark Focus Mode)**，降低长时间注视屏幕的眼部疲劳，同时提高数字对比度。

*   **主色 (Background)**: `#0F172A` (深蓝灰，沉稳，利于专注)
*   **辅色 (Surface/Card)**: `#1E293B` (卡片背景，区分层级)
*   **强调色 (Primary Action)**: `#3B82F6` (亮蓝，用于按钮、进度条)
*   **成功色 (Success)**: `#10B981` (翠绿，用于正确反馈)
*   **错误色 (Error)**: `#EF4444` (珊瑚红，用于错误反馈)
*   **文本色 (Text)**: `#F8FAFC` (主文本), `#94A3B8` (次要文本)

## 视觉风格
**现代极简主义 (Modern Minimalist) + 微科技感**

1.  **字体选择**：
    *   **数字**：使用等宽字体 (Monospace)，如 `Roboto Mono` 或 `JetBrains Mono`。确保数字宽度一致，避免视觉跳动，便于快速扫描记忆。
    *   **UI 文本**：使用无衬线字体 (Sans-serif)，如 `Inter` 或 `Noto Sans SC`，保证清晰度。
2.  **视觉元素**：
    *   **卡片风格**：轻微圆角 (12px)，无描边，使用柔和的环境光阴影区分层级。
    *   **动效**：数字出现/消失使用 `Fade & Scale` (淡入淡出 + 轻微缩放)，过渡时间控制在 0.3s 内，保持流畅不拖沓。
    *   **去装饰化**：移除所有与核心玩法无关的装饰图案，保持界面“冷静”。

## 交互设计
强调**即时性**与**多感官反馈**，降低认知负荷。

1.  **记忆阶段反馈**：
    *   数字显示前，屏幕轻微变暗，聚焦中心。
    *   数字出现时伴随轻微的“ popping"音效（可关闭）。
    *   倒计时结束，数字快速模糊后消失，提示“请输入”。
2.  **输入阶段反馈**：
    *   **按键按下**：键盘按钮有下压态 (Active State)，颜色变深。
    *   **输入正确**：输入框内数字高亮显示。
    *   **输入错误**：设备触发**短促震动 (Haptic Feedback)**，输入框轻微左右摇晃 (Shake Animation)，数字变红。
    *   **提交成功**：全屏绿色闪光脉冲，播放成功音效。
3.  **防误触设计**：
    *   记忆阶段禁用所有输入控件。
    *   结果页“下一关”按钮需有 0.5s 冷却，防止连点。

## 移动端适配
采用**响应式弹性布局**，优先适配竖屏单手操作。

1.  **安全区域 (Safe Area)**：
    *   顶部避开刘海/灵动岛，底部适配 Home Indicator，确保内容不被遮挡。
2.  **拇指热区优化**：
    *   数字键盘高度不低于屏幕的 35%，单个按键高度 ≥ 56pt，间距 ≥ 12pt，方便盲打。
    *   核心操作按钮（如“确认”）放置在右下角拇指最易触及区域。
3.  **横屏适配**：
    *   若用户旋转屏幕，布局自动切换为左右结构（左侧数字/输入，右侧键盘），保持视觉比例不变。
4.  **性能优化**：
    *   所有动效使用 CSS Transform 或原生动画 API，确保 60fps 流畅度。
    *   适配不同 DPI，数字边缘锐利无锯齿。
